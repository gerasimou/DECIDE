/**				Using PrismServer	 				     ***/
/***********************************************************************************************/

1) Extract the zip file (if it’s zipped)


/***********************************************************************************************/
/**				Starting PrismServer instances 				     ***/
/***********************************************************************************************/

To start the server side of the application which runs Prism probabilistic model checker under the hood, 
you need to do the following:

1) Use terminal to navigate to "PrismServer" directory

2) Type “./launch.sh -sX”  (without the quotes)			
	where X is the number of server instances you want to start
	For example, if you want to start 3 server instances you will type: ./launch.sh -s3
	Note: this command will start X server instances and assign automatically listening ports


/***********************************************************************************************/
/**				Stopping PrismServer instances				     ***/
/***********************************************************************************************/

Prism server instances will be killed automatically as soon as the MOOS application is killed.
However, in order to be on the safe side you can type the following command to kill all the active
server instances

1) Use terminal to navigate to "PrismServer" directory

2) Type “./clean.sh -k” (without the quotes)


/***********************************************************************************************/
/**				Cleaning data generated by PrismServer instances	     ***/
/***********************************************************************************************/

To clean data generated by Prism server instances you can type the following command

1) Use terminal to navigate to "PrismServer" directory

2) Type “./clean.sh -s” (without the quotes)
